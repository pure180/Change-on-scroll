How to use this plugin

Insert the plugin to your sourcecode

To activate the plugin, you will need the latest version of jQuery, I recommend version 2.1.4. Just take the Google CDN and add it before the closing </body>.

Then download the latest version of this plugin here. Copy the JS-files to your HTML Project, for example to your "/js" folder and add it before the previously pasted Google CDN in to your source code

It then looks something like this:

<script src="https://ajax.googleapis.com/ajax/libs/jquery/2.1.4/jquery.min.js"></script>
<script src="js/jquery.changeonscroll.min.js"></script>

Activate this plugin via data attributes

You can use this plugin purely through the markup API without writing a single line of JavaScript. This way should be your first consideration when using this plugin.

To enable the functionality on scroll, select your desired HTML element which will be changed and set the necessary "data-attributes". For example: you like that an absolute positioned element moves from left to right, than your DOM element should look like the following example.

<div data-spy="scroller" data-style="left" data-start="0" data-end="500" data-top="0" data-bottom="500"></div>

Activate this plugin via JavaScript

But no worries, you can also activate it using the conventional way as you are used to from jQuery. However, I guess you are a bit limited, unless you want to use one option setup for many DOM Elements.

var mysettings = jQuery.fn.changeonscroll.DEFAULTS {
  style : 'left', // CSS - Style property
  start : 0, // Start - Distance to top/left of start changing CSS Values
  end : 500, // End - Distance to top/left of end changing CSS Values
  top : 0, // Beginn of triggering event
  bottom : 500, // End of triggering event
}
jQuery('.myelements').changeonscroll(mysettings)

Plugin Options

You can use all Plugin options as data attributes through the markup API. Just add your preferred option to the affected element or use JavaScript as usual.

Beware, setting the respective data attributes will override any default options which are already set in JavaScript.

Overview of all possible options

Options Properties
JavaScript markup API

selector

data-spy="scroller"

This markup API initializes the plugin on the affected element and it starts to listen.

style

data-style=""

The preferred CSS style, which will be assigned to the element when scrolling. You can choose one or more properties separated by commas.

In case of using multiple properties, you need to set the same amount of them also in:

  • start or data-start=""
  • end or data-end=""

Unless you don't set them, this plugin won't work as expected.

Single property example:
<div data-spy="scroller" data-style="left" data-start="0" data-end="500"></div>

Multiple properties example:
<div data-spy="scroller" data-style="left,top" data-start="0,10" data-end="500,1000"></div>

Compatible CSS properties as of version 2.0.0

For now, you can use all integer CSS properties.

Positions:
top, left, bottom, right, z-index
Margins:
margin-top, margin-right, margin-bottom, margin-left
Paddings:
padding-top, padding-right, padding-bottom, padding-left
Dimensions:
height, min-height, max-height, width, min-width, max-width
Borders:
border-width, border-top-width, border-right-width, border-bottom-width, border-left-width, border-radius, border-top-left-radius, border-top-right-radius, border-bottom-left-radius, border-bottom-right-radius
Fonts:
font-size, font-size-adjust, font-weight (as a numerical value), line-height, letter-spacing, text-indent
Miscellaneous:
opacity

top

data-top=""

Top trigger, here the plugin begins to listen on the scroll event and starts to calculate the percentage position of the affected element, it overrides the given start value and changes the predefined class scroller_before to scroller_while.

This option can be an integer Value of your choice or an unique identifier like an ID.

For example:

  • If you set top : 500 or data-top="500", the plugin starts to trigger the scroll event when $(window).scrollTop() or $(window).scrollLeft() reached 500px from the top or left edge of the browser window.
  • If you set top : '#topElement' or data-top="#topElement", the plugin starts to trigger the scroll event when $(window).scrollTop() or $(window).scrollLeft() reached the top or left edge of the given element.
  • If you leave this blank, this plugins sets the top trigger to the top or left edge of the affected element. Depending on your given configuration.

bottom

data-bottom=""

Bottom trigger, here the plugin ends to listen on the scroll event and stops to calculate the percentage position of the affected element, it sets the given end value and changes the previously specified class scroller_while to scroller_after.

This option can be an integer Value of your choice or an unique identifier like an ID.

For example:

  • If you set bottom : 1000 or data-bottom="1000", the plugin stops to trigger the scroll event when $(window).scrollTop() or $(window).scrollLeft() reached 1000px from the top or left edge of the browser window.
  • If you set bottom : '#bottomElement' or data-bottom="#bottomElement", the plugin stops to trigger the scroll event when $(window).scrollTop() or $(window).scrollLeft() reached the top or right edge of the given element.
  • If you leave this blank, this plugins sets the bottom trigger to the bottom or right edge of the affected element. Depending on your given configuration.

start

data-start=""

Starting value of each CSS propertie. You can set one or multiple values separated by commas for the given CSS style properties in style or data-style="".

In case of using multiple properties, you need to set the same amount of them also in:

  • style or data-style=""
  • end or data-end=""

Unless you don't set them, this plugin won't work as expected.

Single property example:
<div data-spy="scroller" data-style="left" data-start="0" data-end="500"></div>

Multiple properties example:
<div data-spy="scroller" data-style="left,top" data-start="0,10" data-end="500,1000"></div>

This value is needed to calculates the percentage result in the relation to

  • the top or left edge of the browser window $(window).scrollTop() or $(window).scrollLeft(),
  • this starting value (start, data-start=""),
  • the endinging value (end, data-end=""),
  • the factor (factor, data-factor="").

end

data-end=""

Ending value of each CSS propertie. You can set one or multiple values separated by commas for the given CSS style properties in style or data-style="".

In case of using multiple properties, you need to set the same amount of them also in:

  • style or data-style=""
  • start or data-start=""

Unless you don't set them, this plugin won't work as expected.

Single property example:
<div data-spy="scroller" data-style="left" data-start="0" data-end="500"></div>

Multiple properties example:
<div data-spy="scroller" data-style="left,top" data-start="0,10" data-end="500,1000"></div>

This value is needed to calculates the percentage result in the relation to

  • the top or left edge of the browser window $(window).scrollTop() or $(window).scrollLeft(),
  • the starting value (start, data-start=""),
  • this endinging value (end, data-end=""),
  • the factor (factor, data-factor="").

reverse

data-reverse=""

Set this option if you want to calculate the percentage position of this element in reverse.

factor

data-factor=""

With this option you can multiply the result of the percentage calculation by this value.

beforeClass

data-beforeClass=""

With this option you can change the class name which initialize before this plugin triggers the scroll event.

Default: scroller_before

whileClass

data-whileClass=""

With this option you can change the class name which initialize while this plugin triggers the scroll event.

Default: scroller_while

afterClass

data-afterClass=""

With this option you can change the class name which initialize after this plugin triggers the scroll event.

Default: scroller_after

Plugin Events

This plugin provides a few events for hooking into the changeonscroll functionality.

Event type Description

before.changeonscroll

This event fires before this plugin triggers the scroll event.

jQuery('#element').on('before.changeonscroll', function(){
   // do something before triggering scroll 
})

inside.changeonscroll

This event fires while this plugin triggers the scroll event.

jQuery('#element').on('inside.changeonscroll', function(){
   // do something while triggering scroll 
})

after.changeonscroll

This event fires after this plugin triggers the scroll event.

jQuery('#element').on('after.changeonscroll', function(){
   // do something after triggering scroll 
})
